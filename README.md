# Carwash-System

A Laravel-based Car Wash Booking & Product Purchase System with time-slot management, online payments, and role-based dashboards.

## Features
- Online service booking with date & time slot control
- Slot availability management (prevents duplicate booking)
- Product purchase with pickup or delivery option
- Stripe and ToyyibPay (FPX) payment integration
- Customer, Staff, and Admin dashboards
- Booking, payment, and purchase management

## Technology Stack
- Laravel (PHP Framework)
- MySQL Database
- Blade Template Engine
- Stripe Payment Gateway
- ToyyibPay FPX Payment Gateway

## System Roles
- **Customer**: Book services, purchase products, make payments
- **Staff**: Manage bookings, view payments, update status
- **Admin / Owner**: Manage services, products, staff, and reports

## Purpose
This system was developed as an academic project to demonstrate a real-world car wash management platform with online booking and e-commerce functionality.
